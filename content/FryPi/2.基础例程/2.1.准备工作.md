# <center>准备工作</center>

我们的目标就是让大家快速的使用HAL库进行软件开发，上手项目，主打一个快速上手，所以大家一定要需要不懂的，去问deepseek或者网上冲浪学习~

### 1.拉取仓库

首先需要把仓库的代码下载下来，可以直接到仓库下载安装包，但是更推荐使用`git clone`下拉下来，方便以后拉取更新的代码。

```sh
git clone https://gitee.com/kingham/FryPi.git
```

### 2.原理图

可以去到立创开源打开硬件工程可以得到原理图和PCB，方便进行参考。这里仅放一个pin-map图方便查阅。

### 3.开发环境

需要一个下载器例如`ST-Link`，同时电脑上需要的基本的开发环境有：

- keil MDK
- CubeMX
- STM32 ST-LINK Utility
- Vscode (MDK插件等)

详细的安装详见网络上的教程，安装包也随处可以找到~

### 4.如何烧录

将ST-link插到开发板的SWD下载口, 找到连接好`3V3`, `GND`, `DIO`, `CLK`, 然后在keil中进行烧录即可。如果只是验证demo, 直接将工程中生成的`.hex`文件，拖入`ST-LINK Utility`中，然后点击插头图标`connect to target`连接上开发板，然后再点击白色的图标`program verify`即可下载。
