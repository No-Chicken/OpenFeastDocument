# <center>介绍</center>

**Power-Pico** 是一款专为电子发烧友，嵌入式开发者和工程师，设计的便携测量工具，用于测量**uA**级别微小电流。同时也支持**PPS诱骗**，在外出时只有快充充电头的情况下，也能充当可调电源！配合 **Power-Pico Client 客户端**，它可以对Power-Pico的**波形监控**与**在线升级**。

## <center>产品功能</center>

<div style="display:grid;grid-template-columns:repeat(auto-fit, minmax(280px, 1fr));gap:20px;margin-bottom:50px">

<div style="padding:20px;border-radius:20px;border:1px solid">
    <h3 style="margin:0 0 10px 0;font-size:1.2em">🔋 实时监测</h3>
    <p style="margin:0;font-size:1em"><b>10kHz</b>采样频率，<b>12位ADC</b>采样，ADC+DMA通过<b>高速USB</b>发送数据到上位机实时监控。</p>
</div>


<div style="padding:20px;border-radius:20px;border:1px solid">
    <h3 style="margin:0 0 10px 0;font-size:1.2em">👁️ 优雅客户端</h3>
    <p style="margin:0;font-size:1em">基于<b>PySide6</b>的优雅上位机，支持 <b>中/英/日</b> 多种语言，可以<b>实时监控数据</b>，可以方便地对Power-Pico进行<b>在线升级</b>。</p>
</div>

<div style="padding:20px;border-radius:20px;border:1px solid">
    <h3 style="margin:0 0 10px 0;font-size:1.2em">🚀 PPS协议诱骗</h3>
    <p style="margin:0;font-size:1em">接口全为<b>Type-C</b>接口，通用性拉满。支持通用PD快充协议，<b>PPS诱骗</b>，可充当<b>可调电源</b>。</p>
</div>

<div style="padding:20px;border-radius:20px;border:1px solid">
    <h3 style="margin:0 0 10px 0;font-size:1.2em">😍 全开源</h3>
    <p style="margin:0;font-size:1em">硬件软件都开源！发烧友与极客朋友们，可自由更改UI界面与DIY，原理透明，方便自由探索学习！</p>
</div>
