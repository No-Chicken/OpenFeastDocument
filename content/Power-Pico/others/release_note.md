# <center>版本更新说明</center>

注意，固件版本与硬件版本对应，例如V1硬件对应的固件版本为V1.X.X。硬件改动一般为大改动。

## 硬件更新

#### V1
2026.02.08
* 加入多个电容
* 取消串口CH340, 改用速度更快更稳的USB

2025.12.18
* 上传硬件
* 更新INA199为1NA190，防止过大的uA漏电流
* 串口接口处电源并联多个电容
